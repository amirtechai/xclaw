# `xclaw setup`

Initialize `~/.xclaw/xclaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- CLI onboarding: [Onboarding (CLI)](/start/wizard)

## Examples

```bash
xclaw setup
xclaw setup --workspace ~/.xclaw/workspace
```

To run onboarding via setup:

```bash
xclaw setup --wizard
```
